  1. 시스템 개요
================================================================================

  프린스리브(Prince-Leave)는 우리 회사 직원들이 연차, 반차, 반반차를
  간편하게 신청하고 관리할 수 있는 웹 기반 시스템입니다.

  별도 앱 설치가 필요 없으며, 스마트폰이나 PC의 인터넷 브라우저에서
  바로 사용할 수 있습니다.

  [주요 기능]
  - 연차 / 반차 / 반반차 신청 및 취소
  - 캘린더에서 전 직원 일정 확인
  - 내 연차 잔여일, 사용 이력 조회
  - 신청 시 대표님께 승인 메시지 자동 생성

  ┌─────────────────────────────────────────────────┐
  │  우리 회사는 연차사용촉진을 권장하고 있습니다.   │
  │  팀원간의 일정확인후 자유롭게 사용바랍니다.       │
  └─────────────────────────────────────────────────┘

  2. 접속 방법
================================================================================

  [방법 1] 배포 URL 접속 (권장)
  ─────────────────────────────
  브라우저 주소창에 아래 주소를 입력하세요.

    * 실제 주소는 관리자에게 확인하세요.

  [방법 2] 즐겨찾기 / 홈 화면 추가
  ─────────────────────────────────
  자주 사용하시는 분은 즐겨찾기에 추가하거나,
  스마트폰의 경우 "홈 화면에 추가"를 하시면 편리합니다.

    - iPhone : Safari > 공유 버튼 > 홈 화면에 추가
    - Android : Chrome > 점 3개 메뉴 > 홈 화면에 추가

  [권장 브라우저]
  - Chrome (최신 버전)
  - Safari (최신 버전)
  - Edge (최신 버전)

  4. 연차 신청 방법 (4단계)
================================================================================

  ──────────────────────────
  STEP 1. 연차 신청 버튼 클릭
  ──────────────────────────

  화면 하단의 [연차 신청] 버튼을 누릅니다.
  연차 신청 팝업 창이 열립니다.

  ┌─────────────────────────────────────────────────┐
  │  우리 회사는 연차사용촉진을 권장하고 있습니다.   │
  │  팀원간의 일정확인후 자유롭게 사용바랍니다.       │
  └─────────────────────────────────────────────────┘
  * 위 안내 문구가 신청 화면 상단에 표시됩니다.


  ──────────────────────────
  STEP 2. 신청자 선택
  ──────────────────────────

  본인의 이름을 선택합니다.
  직원 목록이 이름/직급 2줄 형태로 표시되어 쉽게 찾을 수 있습니다.

    예)  도승지
         실장

         정훈필
         대리


  ──────────────────────────
  STEP 3. 휴가 유형 선택
  ──────────────────────────

  아래 3가지 유형 중 하나를 선택합니다.

    [연차]     하루 전체 휴가       (1일 차감)
    [반차]     반일 휴가            (0.5일 차감)
    [반반차]   2시간 단위 휴가      (0.25일 차감)

  * 반차 선택 시 : 오전(AM) / 오후(PM) 중 선택
  * 반반차 선택 시 : 2시간 단위 시간대 선택


  ──────────────────────────
  STEP 4. 날짜 선택 및 신청
  ──────────────────────────

  사용할 날짜를 선택합니다.
  여러 날을 동시에 선택할 수 있습니다 (연차의 경우).

  선택 완료 후 [신청하기] 버튼을 누르면 신청이 완료됩니다.

  ┌──────────────────────────────────────────────┐
  │  신청 완료 시 승인 메시지가 자동으로           │
  │  클립보드에 복사됩니다!                        │
  │                                              │
  │  메시지 예시:                                  │
  │  "대표님, 2026-02-19 에 연차를               │
  │   사용하려고 합니다. 승인부탁드립니다"         │
  └──────────────────────────────────────────────┘

  * 신청 후 반드시 하단의 [저장하기] 버튼을 눌러주세요!


================================================================================
  5. 연차/반차/반반차 취소 방법
================================================================================

  이미 신청한 연차를 취소해야 할 경우 아래 절차를 따르세요.

  ──────────────────────────
  STEP 1. 내 이름 클릭
  ──────────────────────────

  하단 직원 현황에서 본인의 이름을 클릭합니다.
  직원 상세 팝업이 열립니다.


  ──────────────────────────
  STEP 2. 사용 이력 확인
  ──────────────────────────

  팝업 안에서 본인의 사용 이력 목록을 확인합니다.

    예)  2026-01-08  연차    [취소]
         2026-01-16  반반차  [취소]


  ──────────────────────────
  STEP 3. 취소 버튼 클릭
  ──────────────────────────

  취소하고 싶은 항목 옆의 [취소] 버튼을 누릅니다.
  확인 팝업이 뜨면 [확인]을 눌러 취소를 확정합니다.


  ──────────────────────────
  취소 후 자동 처리 내용
  ──────────────────────────

  - 잔여 연차가 자동으로 복구됩니다.
      연차 취소 → +1일 복구
      반차 취소 → +0.5일 복구
      반반차 취소 → +0.25일 복구

  - 캘린더에서 해당 일정이 즉시 삭제됩니다.

  - 취소 메시지가 자동 복사됩니다.
      예) "대표님께 2026-01-08 연차 사용을 취소합니다"

  * 취소 후 반드시 [저장하기] 버튼을 눌러주세요!

  6. 내 연차 현황 확인 방법
================================================================================

  [방법 1] 메인 화면 직원 카드
  ─────────────────────────────

  메인 화면 하단에서 본인의 카드를 확인합니다.

    ┌──────────────────────┐
    │  도승지 실장          │
    │                      │
    │  총연차: 12.5일      │
    │  사용:    1.25일     │
    │  잔여:   11.25일     │
    │                      │
    │  ████████░░  90%     │  ← 사용률 게이지
    └──────────────────────┘


  [방법 2] 상세 팝업 (이름 클릭)
  ─────────────────────────────

  본인 이름을 클릭하면 상세 정보가 표시됩니다.

  확인 가능한 정보:
  - 총 부여일
  - 총 사용일
  - 잔여일
  - 연차 갱신월
  - 사용 이력 전체 목록 (날짜, 유형, 상세)

  7. 캘린더 보는 법
================================================================================

  [색상 안내]
  ───────────

    빨간색 날짜    →  공휴일 (신정, 설날, 추석, 광복절 등)
    파란색 날짜    →  토요일
    회색 날짜      →  일요일
    노란 하이라이트 →  오늘 날짜

  [직원 일정 표시]
  ─────────────────

  각 직원은 고유 색상이 지정되어 있어요.

    도승지 실장  →  보라색 (■)
    정훈필 대리  →  빨간색 (■)
    김영훈 대리  →  주황색 (■)
    유시은 사원  →  초록색 (■)
    지영은 사원  →  보라색 (■)

  연차가 신청된 날짜에는 해당 직원의 색상 마커와
  이름이 캘린더 셀 안에 표시됩니다.


  [월 이동]
  ──────────

  캘린더 상단의 [◀ 이전달] [다음달 ▶] 버튼으로
  다른 달의 일정을 확인할 수 있습니다.


  [공휴일 목록 (2026년)]
  ─────────────────────

    01/01 (목)  신정
    02/16 (월)  설날 연휴
    02/17 (화)  설날
    02/18 (수)  설날 연휴
    03/01 (일)  삼일절
    03/02 (월)  대체공휴일 (삼일절)
    05/05 (화)  어린이날
    05/24 (일)  부처님 오신 날
    05/25 (월)  대체공휴일 (부처님 오신 날)
    06/03 (수)  지방선거일 (임시공휴일)
    06/06 (토)  현충일
    07/17 (금)  제헌절 (18년 만에 공휴일 복원)
    08/15 (토)  광복절
    08/17 (월)  대체공휴일 (광복절)
    09/24 (목)  추석 연휴
    09/25 (금)  추석
    09/26 (토)  추석 연휴
    10/03 (토)  개천절
    10/05 (월)  대체공휴일 (개천절)
    10/09 (금)  한글날
    12/25 (금)  크리스마스

  * 관리자가 별도 추가한 공휴일도 캘린더에 표시됩니다.


================================================================================
  8. 승인 메시지 전달 방법
================================================================================

  연차를 신청하면 아래 형식의 메시지가 자동으로
  클립보드에 복사됩니다.

  ┌──────────────────────────────────────────────┐
  │  대표님, 2026-02-19 에 연차를                │
  │  사용하려고 합니다. 승인부탁드립니다          │
  └──────────────────────────────────────────────┘

  [메시지 전달 방법]

  1) 연차 신청 완료 후 "메시지가 복사되었습니다" 확인
  2) 카카오톡 등 메신저 앱을 엽니다
  3) 대표님 대화방에서 [붙여넣기] (Ctrl+V 또는 길게 누르기)
  4) 전송!

  * 여러 날을 동시 신청한 경우, 모든 날짜가 메시지에 포함됩니다.

  * 취소 시에도 취소 메시지가 자동 복사됩니다.
    예) "대표님께 2026-02-19 연차 사용을 취소합니다"

  9. 주의사항
================================================================================

  [저장 관련]
  ────────────

  ★ 가장 중요한 규칙 ★

  연차 신청, 취소 등 변경 작업 후에는
  반드시 화면 하단의 [💾 저장하기] 버튼을 눌러주세요!

  저장하지 않고 브라우저를 닫거나 새로고침하면
  변경 내용이 사라질 수 있습니다.

  저장이 완료되면 "저장 완료!" 알림이 표시됩니다.


  [신청 관련]
  ────────────

  - 공휴일과 주말에는 연차 신청이 불가합니다.
  - 잔여 연차가 0일이면 추가 신청이 불가합니다.
  - 같은 날짜에 중복 신청은 불가합니다.
  - 신청 전 캘린더에서 팀원 일정을 먼저 확인해 주세요.


  [브라우저 관련]
  ────────────────

  - 브라우저의 "사이트 데이터 삭제" 또는 "캐시 삭제"를
    하면 저장된 연차 데이터가 삭제될 수 있습니다.
  - 시크릿 모드(사생활 보호 모드)에서는 데이터가 저장되지
    않으므로 일반 모드로 사용해 주세요.
  - 다른 기기(예: 회사PC ↔ 스마트폰)에서는 데이터가
    공유되지 않습니다. (Firebase 동기화 시 공유 가능)


================================================================================
  10. 자주 묻는 질문 (FAQ)
================================================================================

  ────────────────────────────────────────────
  Q1. 새로고침했더니 신청한 내용이 사라졌어요!
  ────────────────────────────────────────────

  A. 신청 후 [💾 저장하기] 버튼을 누르지 않은 경우입니다.
     모든 변경 후에는 반드시 저장 버튼을 눌러주세요.
     저장 완료 시 "저장 완료!" 알림이 뜹니다.


  ────────────────────────────────────────────
  Q2. 반차와 반반차의 차이가 뭔가요?
  ────────────────────────────────────────────

  A. 차감 일수가 다릅니다.

     연차   = 하루 전체 (1일 차감)
     반차   = 반일, 오전 또는 오후 선택 (0.5일 차감)
     반반차 = 2시간 단위 (0.25일 차감)


  ────────────────────────────────────────────
  Q3. 승인 메시지가 복사되지 않아요.
  ────────────────────────────────────────────

  A. 일부 브라우저에서 클립보드 권한이 차단된 경우입니다.
     브라우저 주소창 왼쪽의 자물쇠 아이콘을 눌러
     "클립보드" 권한을 "허용"으로 변경해 주세요.

     그래도 안 되면 신청 완료 화면에 표시된 메시지를
     직접 길게 눌러 복사하셔도 됩니다.


  ────────────────────────────────────────────
  Q4. 잔여 연차가 실제와 다르게 표시돼요.
  ────────────────────────────────────────────

  A. 관리자에게 문의해 주세요.
     관리자가 직원 상세 화면에서 연차 갱신 및
     잔여일 수정을 진행할 수 있습니다.


  ────────────────────────────────────────────
  Q5. 다른 직원의 연차를 대신 신청할 수 있나요?
  ────────────────────────────────────────────

  A. 기술적으로는 가능하지만, 반드시 본인이 직접
     신청하는 것을 원칙으로 합니다.
     부득이한 경우 관리자에게 요청해 주세요.


  ────────────────────────────────────────────
  Q6. 공휴일인데 캘린더에 표시가 안 돼요.
  ────────────────────────────────────────────

  A. 관리자가 추가하지 않은 임시공휴일일 수 있습니다.
     관리자에게 공휴일 추가를 요청해 주세요.


  ────────────────────────────────────────────
  Q7. 연차 갱신월이 되면 어떻게 되나요?
  ────────────────────────────────────────────

  A. 연차 갱신월이 되면 관리자가 이전 연차 내용을
     초기화하고 새로운 연차를 부여합니다.
     갱신 후에는 사용 이력이 리셋되고
     새로운 총연차가 반영됩니다.

     각 직원의 갱신월은 입사일 기준으로 다릅니다.
     본인의 갱신월은 상세 화면에서 확인할 수 있습니다.


  ────────────────────────────────────────────
  Q8. 스마트폰에서도 사용할 수 있나요?
  ────────────────────────────────────────────

  A. 네, 모바일 브라우저에서 동일하게 사용 가능합니다.
     Chrome 또는 Safari를 권장합니다.
     홈 화면에 추가하시면 앱처럼 편리하게 쓸 수 있어요.


  ────────────────────────────────────────────
  Q9. 과거에 사용한 연차 이력도 볼 수 있나요?
  ────────────────────────────────────────────

  A. 네, 본인 이름을 클릭하면 열리는 상세 팝업에서
     전체 사용 이력을 날짜순으로 확인할 수 있습니다.


  ────────────────────────────────────────────
  Q10. 관리자 기능은 어떻게 사용하나요?
  ────────────────────────────────────────────

  A. 관리자 기능은 별도 비밀번호가 필요합니다.
     관리자 권한이 필요한 경우 담당자에게 문의해 주세요.

     관리자 기능에는 다음이 포함됩니다:
     - 신규 직원 추가
     - 연차 갱신 및 부여일 수정
     - 공휴일 추가/삭제
     - 직원 연차 이력 초기화
